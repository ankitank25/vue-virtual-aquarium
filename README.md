# vue-calculator

A Virtual Aquarium app build using Vue3, TailwindCSS.

## Clone Repository

```sh
git clone https://github.com/ankitank25/vue-virtual-aquarium.git
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
